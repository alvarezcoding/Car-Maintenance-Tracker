# Car-Maintenance-Tracker

A command-line application written in Python that helps users track vehicle maintenance and service intervals. 
The program stores data using JSON and calculates when maintenance is due based on mileage.

## Features
  - Add and track vehicle service records
  - Automatically calculate maintenance intervals based on mileage
  - Persistent data storage using JSON
  - Simple user-friendly command line interface

## Software
  - Python
  - JSON

## How to Run
  1. Verify you have Python installed
  2. Download the files
  3. Navigate to the correct project folder in terminal
  4. Run the program:
     python3 car_tracker.py

## Example of Usage:
  - Enter current mileage: 50000 
  - Add service record? (yes/no): yes 
  - Service added successfully! 
  - Next oil change due at 55000 miles
